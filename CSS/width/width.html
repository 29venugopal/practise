WIDTH OF CSS

Syntax :- Topics about the Width.

Subject :-

* The CSS Width properties are used to set the height of an element.

* The Width properties doesnot include padding,borders, or margins.

* It sets the Width of the area inside the padding, border, and margin of the element.

CSS Width values

1. auto - This is default. The browser calculates the height .

2. [ % ] the height in percent of the containing block.

3. length - defines the height in px, cm, .... et.

4. Initial - Sets the height to its default value.

5.Inherit - The height will be inherited from its parent value.
