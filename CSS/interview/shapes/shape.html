SHAPES OF CSS

Syntax :- Topic about shapes concept

Subject :-

* CSS is capable of making of all sort of shapes.

* Square and rectangle are easy, as they are the natural shapes of the web.

* Add a width and height and you will get the exact shape of any thing.

* Add border and radius and you can round the shape.

* Shapes are like square, Rectangle, Circle, Oval, Triangle right,left,up and down,....Etc

* By using the clip path code we can many and many shapes in this way.
